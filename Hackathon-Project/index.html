About the Project

Students at KLH lose break time standing in canteen queues. This system lets them pre-order meals online, get a QR-coded token, and collect food without waiting. The admin side handles orders, menus, and billing in real time.

😓 The Problem

  • Long queues during break time
  • Manual order tracking errors
  • No meal availability info
  • Cash delays at the counter

✅ Our Solution

  • Pre-order from anywhere on campus
  • QR token for contactless pickup
  • Real-time order status updates
  • Mock payment integration

Key Features

🍛 Menu Display

Browse daily menu with veg/non-veg filters and availability.

📝 Token Form

Select item, quantity, and add special instructions.

🥗 Dietary Filter

Filter by vegetarian or non-vegetarian preference.

🚦 Daily Limit

Max 3 orders per student per day with a progress bar.

💳 Payment Mock

Simulate UPI, card, or cash with GST breakdown.

🎫 QR Token

Unique QR code generated per order for pickup.

📊 Billing Dashboard

Admin revenue stats and transaction history.

📋 Order History

Students view past orders and spending anytime.

🔔 Notifications

Alerts for order status and canteen updates.

⚙️ Admin Panel

Manage menu, mark orders ready, control settings.
